Giới thiệu

Lopza.js là một thư viện JavaScript thuần (Vanilla JS) mạnh mẽ, được thiết kế đặc biệt để tạo slider mượt mà và linh hoạt trong giao diện người dùng. Lopza.js cung cấp cách đơn giản để thiết lập và điều khiển carousel, slideshow, và image slider trong bất kỳ ứng dụng web nào, hỗ trợ nhiều tính năng như autoplay, navigation, pagination, draggable, loop, và hiệu ứng chuyển động tùy chỉnh.

Đặc điểm nổi bật

  • Nhẹ và nhanh: Lopza.js chỉ ~9KB, không yêu cầu thư viện phụ thuộc nào.
  • Tích hợp linh hoạt: Hoạt động tốt với mọi dự án và framework mà không cần cấu hình phức tạp.
  • Tùy biến cao: Dễ dàng điều chỉnh hành vi, hiệu ứng chuyển động và các tùy chọn như autoplay, loop, direction, draggable, v.v.
  • CSS dễ dàng tùy chỉnh: Thiết kế slider theo phong cách riêng mà không cần chỉnh sửa mã nguồn chính của thư viện.
  • Thân thiện với lập trình viên: Cung cấp callback rõ ràng, giúp dễ dàng kiểm soát sự kiện chuyển slide và mở rộng tính năng theo nhu cầu dự án.

Callback

Tham Số Mô Tả
prevIndex Vị trí (chỉ số) của tab trước đó. Nếu là lần đầu tiên mở tab (không có tab trước), giá trị sẽ là null.
nextIndex Vị trí (chỉ số) của tab hiện tại mà người dùng đã chọn.
prevTab Phần tử HTML của tab-item trước đó. Nếu là lần đầu tiên mở tab (không có tab trước), giá trị sẽ là null.
nextTab Phần tử HTML của tab-item hiện tại mà người dùng đã chọn.
prevPanel Phần tử HTML của tab-panel trước đó. Nếu là lần đầu tiên mở tab (không có panel trước), giá trị sẽ là null.
nextPanel Phần tử HTML của tab-panel hiện tại tương ứng với tab người dùng đã chọn.
tabContainer Phần tử HTML của toàn bộ container chứa danh sách tab-item và tab-panel.

Hiệu ứng

Để áp dụng hiệu ứng chuyển tiếp cho nội dung của các tab, bạn cần tạo các keyframes như ví dụ bên dưới và thêm chúng vào các lớp CSS của phần tử chứa nội dung tab.

css 
.tab-panel { /* class có thể là tên khác trong dự án của bạn*/
  animation: fadeIn 0.3s ease-in-out;
}
/* Animation cho nội dung */
@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
/* Thêm các Animation khác tại đây ...*/

Chú ý: Hướng dẫn CSS đúng cách.

Dưới đây là hướng dẫn về cách tùy chỉnh CSS cho các phần tử tab:

CSS cho nội dung tab: Bạn có thể tùy chỉnh CSS cho các phần tử chứa nội dung tab như bình thường. VD bên dưới là tab-panels và tab-panel

CSS cho tab-item ở trạng thái active: Để thay đổi lớp CSS cho tab-item khi nó ở trạng thái active, bạn cần khai báo class này trong quá trình khởi tạo tab thông qua tùy chọn activeClass, ví dụ: activeClass: "custom-active". Bạn có thể thay đổi tên lớp này theo ý muốn.

Class custom-active được khởi tạo từ activeClass sẽ xuất hiện như ví dụ bên dưới. Nó biểu thị cho trạng thái khi tab được chọn
